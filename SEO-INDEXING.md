# SEO Indexing Guide

Sitemap submission
1) Generate or update your sitemap to include:
   - https://area-iq.com/journal
   - https://area-iq.com/journal/:slug
2) Submit in Google Search Console:
   - Indexing -> Sitemaps -> Add a new sitemap URL

Check indexing status
- Search Console -> Pages
- Filter for /journal
- Verify indexed and any excluded reasons

Track impressions
- Search Console -> Performance -> Search results
- Filter page path contains /journal
- Compare impressions and clicks over time

AI citation checks
- Google Search: look for AI Overview mentions of /journal pages
- Microsoft Copilot: query the headline and check citations
- Perplexity: query the location headline and check sources

30-60-90 day review
- 30 days: at least a few journal pages indexed
- 60 days: majority indexed, impressions > 0
- 90 days: if indexing is weak, plan SSR or SSG migration
